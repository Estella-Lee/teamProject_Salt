★깃 초기 설정하기 

* git bash 설치하기
* 각자 본인의 컴퓨터에 원하는 위치에 디렉토리를 만들기

해당 폴더를 오른쪽 클릭 후  [git bash here] 눌러서 명령프롬프트 실행

① git config --global user.name "본인계정이름"
② git config --global user.email "본인 이메일 주소"

=> git config --list 입력해서 제대로 입력됐는지 확인하기!

③ git clone https://github.com/Estella-Lee/teamProject_Salt.git

=> git status 확인하는 법 : ls 또는 git status 



★브랜치
① git branch 입력해서 현재 브랜치 확인하기 

② git checkout develop : develop 브랜치로 이동하기 
   => git branch 다시 입력해서 develop 브랜치인지 꼭 확인하기! 

③ git branch 브랜치명  : 새로운 브랜치를 생성함

④ git checkout 브랜치명 : 생성한 branch로 이동
   
⑤ git push --set-upstream origin 브랜치명 : 생성한 브랜치를 깃허브에 푸쉬



★깃 기본 사용법 
1. git add 파일명
2. git commit -m "커밋메세지"
3. git push origin 브랜치명
